## ✨ 무엇을 하는 앱인가요?

**NotionAI Pet**은 바탕화면 한쪽에 항상 떠 있는(Always-on-Top) 귀여운 캐릭터입니다. 여러분의 노션 워크스페이스를 지켜보다가, 페이지가 새로 만들어지거나 내용이 편집되면 **말풍선으로 바로 알려줍니다.** 노션 AI가 긴 문서를 작성하는 동안 다른 작업을 하고 있어도, 펫이 진행 상황을 실시간으로 중계해 줘요.

이런 분께 좋아요:

- 📝 노션 AI에게 문서 작성을 맡겨 두고, **완료되는 순간을 놓치고 싶지 않은 분**
- 👥 팀 워크스페이스에서 **누군가 페이지를 만들거나 고치면 바로 알고 싶은 분**
- 🖥️ 바탕화면에 **귀여운 동반자**를 두고 싶은 분

주요 기능:

- 💬 **실시간 말풍선** — 노션 페이지가 만들어지거나 편집되면 펫이 바로 알려줘요
- ✍️ **작성 내용 미리보기** — 편집 중인 페이지의 새 문장을 말풍선으로 실시간 중계해요
- 🎭 **상태 애니메이션** — 대기 / 작업 중 / 말하기 / 잠자기 (90초 동안 아무 일 없으면 잠들어요)
- 🖱️ **인터랙션** — 드래그로 어디든 이동, 클릭하면 반응 (자고 있으면 깨어나요)
- 📌 **위치 고정** — 우클릭 메뉴에서 위치를 고정하면 실수로 드래그해도 움직이지 않아요
- 🔔 **트레이 상주** — 창을 닫아도 시스템 트레이에서 계속 실행
- 🔐 **안전한 토큰 보관** — Notion 토큰은 OS 보안 저장소(Windows 자격 증명 관리자 / macOS 키체인)에 암호화 저장

## 🌱 노션이 처음이신가요? — 기본 개념 잡기

이 앱을 사용하려면 노션의 몇 가지 개념만 알면 충분합니다. 이미 노션에 익숙하다면 [설치하기](#-설치하기)로 건너뛰세요.

### 노션(Notion)이란?

[Notion](https://www.notion.so)은 메모, 문서, 할 일 목록, 데이터베이스를 한곳에서 관리하는 올인원 워크스페이스 서비스입니다. 무료 계정으로도 이 앱을 사용할 수 있습니다.

### 꼭 알아야 할 3가지 개념

| 개념 | 설명 |
|---|---|
| **워크스페이스 (Workspace)** | 내 노션 콘텐츠 전체가 담기는 공간. 개인용/팀용 워크스페이스를 여러 개 가질 수 있어요. |
| **페이지 (Page)** | 노션의 기본 문서 단위. 페이지 안에 하위 페이지를 계속 만들 수 있어요. |
| **통합 (Integration)** | 외부 앱(이 앱 같은)이 내 워크스페이스를 읽을 수 있게 해 주는 **열쇠**. 통합을 만들면 `ntn_...`으로 시작하는 **토큰(비밀 문자열)**이 발급됩니다. |

### 이 앱이 노션과 연결되는 방식

```
[NotionAI Pet 앱] ──(내 통합 토큰으로)──▶ [Notion API] ──▶ [통합이 연결된 페이지들]
```

1. 여러분이 노션에서 **통합(Integration)** 을 하나 만들고 토큰을 발급받습니다. (무료, 1~2분 소요)
2. 감지하고 싶은 노션 페이지에 그 통합을 **연결**합니다.
3. 앱에 토큰을 입력하면, 앱이 연결된 페이지들의 변화를 감시하기 시작합니다.

> 🔒 **안심하세요**: 이 앱은 여러분의 노션 데이터를 어디에도 전송하지 않습니다. 토큰은 내 컴퓨터의 OS 보안 저장소에만 저장되고, 노션 서버와 직접 통신하는 데만 사용됩니다. 또한 통합에 **읽기 권한만** 주면 되므로, 앱이 노션 내용을 수정하거나 삭제할 수 없습니다.

자세한 발급 방법은 아래 [시작하기](#-시작하기--처음부터-끝까지-따라-하기)에서 그림 그리듯 단계별로 안내합니다.

## 📥 설치하기

> **요구 사항**
> - **Windows**: Windows 10/11 (x64) + WebView2 Runtime (Windows 11은 기본 내장, Windows 10은 설치 시 자동 안내)
> - **macOS**: macOS 11 (Big Sur) 이상 — Apple Silicon / Intel 모두 지원
> - **노션**: 무료 계정이면 충분해요. 노션 AI 구독이 없어도 페이지 활동 감지는 동작합니다.

### 방법 1 — 설치 파일 다운로드 (권장)

1. [**Releases**](https://github.com/developer-snoopy/notionai-pet/releases/latest) 페이지를 엽니다.
2. **Assets**에서 운영체제에 맞는 파일을 내려받아 실행합니다.

   | OS | 파일 | 비고 |
   |---|---|---|
   | Windows (x64) | `notionai-pet_0.4.0_x64-setup.exe` | NSIS 설치 파일 (권장) |
   | Windows (x64) | `notionai-pet_0.4.0_x64_en-US.msi` | MSI 설치 파일 |
   | macOS (Apple Silicon) | `notionai-pet_0.4.0_aarch64.dmg` | M1/M2/M3/M4 |
   | macOS (Intel) | `notionai-pet_0.4.0_x64.dmg` | Intel Mac |

   > 💡 **내 Mac이 Apple Silicon인지 모르겠다면**:  메뉴 → **이 Mac에 관하여**에서 "칩" 항목에 Apple M1/M2/M3/M4가 보이면 Apple Silicon입니다.

3. 설치 안내에 따라 진행하면 앱이 설치됩니다. (Windows: 시작 메뉴 등록, macOS: Applications 폴더로 드래그)

> 💡 **Windows**: 서명되지 않은 앱이라 SmartScreen 경고("Windows의 PC 보호")가 뜰 수 있습니다. **추가 정보 → 실행**을 누르면 설치할 수 있습니다.
>
> 💡 **macOS**: 서명되지 않은 앱이라 처음 실행 시 차단될 수 있습니다. **Finder에서 앱을 우클릭 → 열기**를 선택하거나, 터미널에서 아래 명령으로 격리 속성을 제거하세요.
> ```bash
> xattr -dr com.apple.quarantine /Applications/notionai-pet.app
> ```

### 방법 2 — 명령줄(Command Line)로 설치

```cmd

"%TEMP%\notionai-pet-setup.exe" /S
```

PowerShell:

```powershell
Invoke-WebRequest -Uri "https://github.com/developer-snoopy/notionai-pet/releases/download/v0.4.0/notionai-pet_0.4.0_x64-setup.exe" -OutFile "$env:TEMP\notionai-pet-setup.exe"
& "$env:TEMP\notionai-pet-setup.exe" /S
```

**MSI 설치 파일 (msiexec · 시스템 단위 설치)**


PowerShell (관리자):

```powershell
msiexec /i "$env:TEMP\notionai-pet.msi" /qn
```

**GitHub CLI를 사용하는 경우**


제거는 **설정 → 앱 → notionai-pet → 제거** 또는 `msiexec /x "$env:TEMP\notionai-pet.msi" /qn` 으로 할 수 있습니다.

**macOS** — 터미널을 열고 아래를 실행하세요. (Apple Silicon은 `aarch64`, Intel은 `x64`)

```bash
# 다운로드 (Apple Silicon 기준)
curl -L -o /tmp/notionai-pet.dmg \
  "https://github.com/developer-snoopy/notionai-pet/releases/download/v0.4.0/notionai-pet_0.4.0_aarch64.dmg"

hdiutil detach "/Volumes/notionai-pet"

# Gatekeeper 격리 속성 제거 (서명되지 않은 앱)
xattr -dr com.apple.quarantine /Applications/notionai-pet.app
```

제거는 `/Applications/notionai-pet.app`을 휴지통으로 옮기면 됩니다.

## 🚀 시작하기 — 처음부터 끝까지 따라 하기

노션을 처음 쓰는 분도 그대로 따라 하면 되는 전체 과정입니다. 약 5분이면 끝나요.

### 0단계 — 노션 계정 준비 (이미 있다면 건너뛰기)

1. [notion.so](https://www.notion.so)에 접속해 무료 계정을 만듭니다.
2. 워크스페이스가 자동으로 생성되고, 첫 페이지를 만들 수 있습니다.
3. 테스트용으로 페이지를 하나 만들어 두면 좋아요. (좌측 사이드바 → **+ 페이지 추가**)

### 1단계 — Notion Integration Token 발급

이 앱이 노션 워크스페이스를 **읽을 수 있게** 해 주는 열쇠(토큰)를 만드는 단계입니다.

1. 브라우저에서 [notion.so/my-integrations](https://www.notion.so/my-integrations)에 접속합니다. (노션에 로그인된 상태여야 해요)
2. **+ New integration** (새 통합 만들기) 버튼을 클릭합니다.
3. 다음처럼 입력합니다:
   - **Name(이름)**: 자유롭게 — 예: `NotionAI Pet`
   - **Associated workspace(연결할 워크스페이스)**: 펫이 지켜볼 워크스페이스 선택
   - **Type(유형)**: Internal (기본값 그대로)
4. **Save(저장)** 을 누릅니다.
5. 만들어진 통합의 **Capabilities(기능)** 탭에서 **Read content(콘텐츠 읽기)** 권한이 켜져 있는지 확인합니다.
   - *Update/Insert content 권한은 필요 없습니다. 읽기만 있으면 돼요.*
6. **Secrets(시크릿)** 탭에서 **Internal Integration Secret**의 **Show** → **Copy**를 눌러 토큰을 복사합니다.
   - 토큰은 `ntn_...` 또는 `secret_...`으로 시작하는 긴 문자열입니다.
   - ⚠️ 이 토큰은 **비밀번호와 같습니다.** 다른 사람에게 공유하거나 공개된 곳에 붙여 넣지 마세요.

### 2단계 — 통합을 페이지에 연결 (가장 많이 빠뜨리는 단계!)

토큰을 만들었다고 끝이 아닙니다. 노션은 보안상 **통합을 연결해 준 페이지만** 외부 앱이 볼 수 있게 합니다.

1. 노션에서 펫이 지켜보길 원하는 페이지를 엽니다.
2. 페이지 우측 상단의 **`···`** (더 보기) 메뉴를 클릭합니다.
3. **연결(Connections)** → **연결 검색** 에서 1단계에서 만든 통합 이름(예: `NotionAI Pet`)을 찾아 클릭합니다.
4. "액세스 허용" 확인 창에서 **확인**을 누릅니다.

> 💡 **꿀팁**: **최상위 페이지에 연결하면 그 아래 모든 하위 페이지도 함께 감지됩니다.** 워크스페이스 전체를 지켜보게 하고 싶다면, 사이드바의 최상위 페이지들에 각각 연결해 주세요.
>
> ⚠️ 연결된 페이지가 하나도 없으면 펫은 아무것도 감지하지 못합니다. "펫이 조용해요"의 원인 1위!

### 3단계 — 앱 로그인

1. 설치한 **NotionAI Pet** 앱을 실행하면 로그인 화면이 나타납니다.
2. 1단계에서 복사한 토큰을 입력란에 붙여 넣고 **로그인**을 누릅니다.
   - 앱이 노션 서버에 토큰이 유효한지 확인하고, 내 워크스페이스 정보를 가져옵니다.
3. **AI 권한 확인** 화면이 나타나면 안내를 읽고 **계속 진행하기**를 누릅니다.
   - 노션 공개 API에는 "AI 기능 사용 가능 여부"를 조회하는 기능이 없어서 자동 확인이 불가능할 뿐, 앱 사용에는 지장이 없습니다.

> 🔒 토큰은 이 기기의 OS 보안 저장소(Windows 자격 증명 관리자 / macOS 키체인)에만 저장되며 외부로 전송되지 않습니다.

### 4단계 — 펫 표시하기

1. 홈 화면에서 **노션 펫 보이기** 토글을 켭니다.
2. 바탕화면에 펫이 나타나며 인사를 건넵니다! 🎉
3. 펫을 드래그해서 마음에 드는 위치로 옮겨 보세요.

### 5단계 — 잘 동작하는지 테스트

1. 노션에서 2단계에 연결해 둔 페이지(또는 그 하위 페이지)를 엽니다.
2. 아무 내용이나 몇 줄 적어 봅니다.
3. 최대 30초 안에 펫이 말풍선으로 편집 소식을 알려주면 성공! ✍️
4. 계속 작성하면 펫이 연필을 들고 "작업 중" 모션과 함께 새로 쓴 내용을 미리보기로 보여줍니다.

> 💡 첫 실행 직후에는 기존 페이지들의 상태(기준선)만 수집하므로, **그 이후에 일어난 변경부터** 알려줍니다.

## 🎮 사용법

### 기본 조작

| 하고 싶은 것 | 방법 |
|---|---|
| 펫 이동하기 | 펫을 드래그해서 원하는 위치로 이동 (위치 고정이 꺼져 있을 때) |
| 펫과 대화하기 | 펫을 클릭하면 랜덤 메시지로 반응 |
| 펫 우클릭 메뉴 | 펫을 우클릭하면 **실행(메인 화면) / 위치 고정 / 종료** 메뉴 표시 |
| 위치 고정하기 | 펫 우클릭 → **📌 위치 고정** 토글. 켜면 드래그해도 움직이지 않고, 다시 끄면 자유롭게 이동 가능 |
| 잠든 펫 깨우기 | 펫을 클릭 |
| 펫 숨기기/보이기 | 홈 화면 토글 또는 트레이 아이콘 우클릭 → **펫 보이기/숨기기** |
| 메인 창 다시 열기 | 트레이 아이콘 우클릭 → **메인 화면 열기** |
| 부팅 시 자동 실행 | 홈 화면 → **시작 시 자동 실행** 토글 |
| 로그아웃 | 홈 화면 하단 **로그아웃** (저장된 토큰도 삭제됩니다) |
| 앱 완전히 종료 | 트레이 아이콘 우클릭 → **종료** 또는 펫 우클릭 → **종료** |

> 💡 메인 창의 ✕ 버튼을 눌러도 앱은 종료되지 않고 트레이(작업 표시줄 오른쪽 아이콘 영역)에 계속 상주합니다. 완전히 끄려면 트레이 아이콘을 우클릭해 **종료**를 선택하세요.

### 펫의 상태 읽는 법

펫의 모습만 봐도 지금 무슨 일이 일어나는지 알 수 있어요.

| 상태 | 모습 | 언제? |
|---|---|---|
| 대기 (idle) | 둥실둥실 떠다니며 ✨ 반짝임 | 평상시 — 노션을 조용히 지켜보는 중 |
| 말하기 (talking) | 말풍선 + 몸 흔들기 | 노션 활동을 감지했거나, 클릭에 반응할 때 |
| 작업 중 (working) | 연필 들고 열심히 ✍️ | 누군가(또는 노션 AI가) 페이지를 쓰고 있는 걸 추적하는 중 |
| 잠자기 (sleeping) | 눈 감고 zzz 💤 | 90초 동안 아무 활동이 없을 때 — 클릭하면 깨어나요 |

## 🔍 노션 활동 감지 방식

펫이 어떻게 노션을 지켜보는지 궁금하다면:

- **30초마다** Notion API(`/v1/search`)를 폴링하여 페이지·데이터베이스의 생성/편집을 감지합니다.
- 편집이 감지된 페이지는 **추적 모드**로 전환되어 **8초 간격**으로 본문을 비교(diff)하고, **새로 작성 중인 내용을 말풍선 미리보기**로 실시간 표시합니다. 추적 중에는 펫이 연필을 들고 작업 중 모션을 보여줍니다. ✍️
- **60초** 동안 변화가 없으면 추적을 멈추고 평상시 감시로 돌아갑니다.
- 첫 실행 시에는 기준선만 수집하고, 이후 변경분부터 말풍선으로 알려줍니다.
- 감지 범위는 **통합이 연결된 페이지**로 한정됩니다 ([2단계](#2단계--통합을-페이지에-연결-가장-많이-빠뜨리는-단계) 참고).
- Notion 공개 API 특성상 편집 주체(AI/사람)는 구분되지 않으며, 연결된 페이지의 모든 변경이 표시됩니다.
- Notion API 요청 한도(초당 3회)를 준수하도록 폴링 주기가 설계되어 있어, 워크스페이스에 부하를 주지 않습니다.

## ❓ 문제 해결 & 자주 묻는 질문

**Q. "유효하지 않은 토큰" 오류가 나요.**
- 토큰 앞뒤에 공백이 들어가지 않았는지, 문자열 **전체**를 복사했는지 확인해주세요.
- [my-integrations](https://www.notion.so/my-integrations)에서 해당 통합이 삭제되지 않고 활성 상태인지 확인해주세요.
- 토큰을 재발급(refresh)했다면 이전 토큰은 무효가 됩니다. 새 토큰으로 다시 로그인하세요.

**Q. 펫이 아무 말도 하지 않아요.**
- 가장 흔한 원인: **통합이 페이지에 연결되지 않았어요.** 노션 페이지에서 `···` → **연결** → 통합 추가를 확인해주세요 ([2단계](#2단계--통합을-페이지에-연결-가장-많이-빠뜨리는-단계)).
- 연결한 페이지(또는 그 하위 페이지)에서 실제로 편집이 일어나야 감지됩니다. 테스트로 페이지에 몇 글자 적어 보세요.
- 첫 실행 직후에는 기준선 수집 때문에 기존 활동은 표시되지 않고, **이후의 새 변경부터** 알려줍니다.
- 폴링 주기가 30초라서 최대 30초 정도 늦게 알려줄 수 있어요.

**Q. "Notion AI 권한을 확인할 수 없습니다"라고 떠요.**
Notion 공개 API에는 AI 기능 활성화 여부를 조회하는 엔드포인트가 없어 자동 확인이 불가합니다. 안내를 확인하고 **계속 진행하기**를 누르면 정상적으로 사용할 수 있습니다. 노션 AI 구독이 없어도 페이지 활동 감지는 동작해요.

**Q. 노션 AI가 쓴 것과 사람이 쓴 것을 구분할 수 있나요?**
아쉽게도 Notion 공개 API는 편집 주체를 알려주지 않아 구분이 불가능합니다. 연결된 페이지의 모든 변경이 표시됩니다.

**Q. 토큰은 안전하게 보관되나요?**
네. 토큰은 OS 보안 저장소(Windows 자격 증명 관리자 / macOS 키체인)에 저장되며, 이 기기 밖으로 전송되지 않습니다. Notion API 호출에만 사용됩니다. 통합에 읽기 권한만 부여했다면 이 앱(또는 토큰이 유출되더라도 해당 토큰)으로는 노션 내용을 수정할 수 없습니다.

**Q. 펫이 화면에서 사라졌어요.**
- 홈 화면의 **노션 펫 보이기** 토글이 켜져 있는지 확인하세요. (트레이 아이콘 우클릭 → **메인 화면 열기**)
- 모니터 구성이 바뀌면(예: 외부 모니터 분리) 펫이 화면 밖에 있을 수 있습니다. 토글을 껐다 켜 보세요.

**Q. 펫이 자꾸 잠들어요. 고장인가요?**
아니요, 정상이에요! 90초 동안 노션 활동이 없으면 펫이 잠듭니다 💤. 클릭하면 깨어나고, 새 활동이 감지되면 스스로 일어나 알려줍니다.

**Q. 회사/팀 워크스페이스에서도 쓸 수 있나요?**
네. 다만 워크스페이스 관리자가 통합 생성을 제한해 둔 경우 관리자에게 요청이 필요할 수 있습니다. 통합은 연결해 준 페이지만 읽을 수 있으므로, 팀에서 공개하기로 한 페이지에만 연결하는 것을 권장합니다.

**Q. 앱을 삭제하면 토큰도 삭제되나요?**
앱 안에서 **로그아웃**을 하면 저장된 토큰이 즉시 삭제됩니다. 앱만 제거했다면 [my-integrations](https://www.notion.so/my-integrations)에서 통합 자체를 삭제해 토큰을 무효화할 수도 있습니다.

## 📚 용어 사전

처음 보는 단어가 있다면 여기서 확인하세요.

| 용어 | 뜻 |
|---|---|
| **통합 (Integration)** | 외부 앱이 노션 워크스페이스에 접근할 수 있게 해 주는 노션의 공식 연결 방식 |
| **토큰 (Token)** | 통합을 만들면 발급되는 비밀 문자열(`ntn_...`). 앱이 노션에 "나 허가받은 앱이야"라고 증명하는 열쇠 |
| **연결 (Connection)** | 특정 노션 페이지에 통합의 접근을 허용하는 것. 페이지 `···` 메뉴에서 설정 |
| **폴링 (Polling)** | 앱이 주기적으로(30초마다) 노션에 "새 소식 있어?"라고 물어보는 방식 |
| **트레이 (System Tray)** | Windows 작업 표시줄 오른쪽(시계 옆) / macOS 메뉴 막대의 작은 아이콘 영역. 앱이 여기 상주해요 |
| **Always-on-Top** | 다른 창들보다 항상 위에 표시되는 창 속성. 펫이 어떤 작업 중에도 보이는 이유 |
| **WebView2** | Windows에서 웹 기반 앱을 구동하는 마이크로소프트의 런타임. Windows 11엔 기본 내장 |
